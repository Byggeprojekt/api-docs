Documents

Request Purpose

Get list of documents which are available for specified user in the specified folder/project.

Request Type

GET

Request Endpoint

/api/v0/document/getdocuments

Request Parameters

Parameter Description
token Authentication token.
projectid Identifier of project.
parentfolderid Identifier of parent folder. Not needed if root level folders are requested.
all If all documents of project should be returned, parameter all should be set as "true". In this case parameter parentFolderId is not needed. Parameter all should not be passed if parentfolderid parameter is passed.

Response Type

json

https://app.byggeprojekt.dk/api/v0/document/getdocuments?token=13a6833433774dda8dd8ef8979d1180c&projectid=1599&parentfolderid=74950

Response Objects and fields

Parameter Description
statusCode Result code of request. Describes if request was successfully executed.
statusMessage Description to statusCode.
body Object with result for request.
documents List with documents objects.

Document object

Parameter Description
id Identifier of document.
name Name of document.
version Version of document.
parentFolderId Identifier of parent folder.
projectId Identifier of project.
format Metadata field.
scale Metadata field.
revision_number Metadata field.
fileSize Size of files in bytes.
uploadDate Date of uploading the version of file.
documentDescription Description of document.
documentCreator Creator of document.
url URL for downloading the file.
