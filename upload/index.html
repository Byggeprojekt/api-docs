Upload

Request Purpose

Uploading document into system.

Request Type

POST

Request Endpoint

/api/v0/auth/upload

Request Parameters

Parameter Description
token Authentication token.
folderId Identifier of folder into which should be uploaded files.
isLastFileInSession Optional. true or false value for pointing if file or files last in upload session. If parameter not signed in request files will be uploaded like last in session
uploadSessionId Optional. Required if some files was uploaded earlier and it is need upload files in the same upload session.

Also to request must be attached file parts.

Response Objects and fields

Parameter Description
statusCode Result code of request. Describes if request was successfully or unsuccessfully executed.
statusMessage Description to statusCode.
body Object with result for request.
uploadSessionId Upload session identifier, should be used for next uploads in the same session.
documents Array of document objects. See document object

Code Message
401 Authentication problem
402 No token parameter
413 Wrong folderid parameter
416 Wrong isLastFileInSession parameter
417 No folderid parameter
421 No files attached
431 Current user have not permissions for uploading into folder
444 Application is not ready, try again after few minutes
