Authentication

Request Purpose

Authenticate user and receive authentication token for access to other methods.

Request Type

GET

Request Endpoint

/api/v0/auth/gettoken

Request Parameters

Parameter Description
login User login.
password User password.

Request Example

https://app.byggeprojekt.dk/api/v0/auth/gettoken?login=iirsoft3&password=password

Response Example

{
    "statusMessage":"Success",
    "statusCode":200,
        "body":{
            "token":"13a6833433774dda8dd8ef8979d1180c",
            "expirationDate":"Jun 3, 2017 9:59:34 AM"
       }
}

Response Objects and fields

Parameter Description
statusCode Result code of request. Describes if request was successfully executed.
statusMessage Description to statusCode.
body Object with result for request.
token Security token, should be used in next requests.
expirationDate Expiration date of token.
